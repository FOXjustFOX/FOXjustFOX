# Project Context: Igor Lis Portfolio

## Project Overview
Personal portfolio website for Igor Lis, a Systems Engineering student. The site features a dark cinematic theme, advanced animations (GSAP, ScrollTrigger), and showcases his projects (PWr Now, AI Clone).

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: TailwindCSS
- **Animations**: GSAP, Lenis, Framer Motion (optional, defaulting to GSAP as requested)
- **Deployment**: Vercel-ready

## Configuration
- **Linting**: `npm run lint`
- **Formatting**: `prettier` (if configured)
- **Dev Server**: `npm run dev`
